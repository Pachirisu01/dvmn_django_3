# Скрипты для работы с электронным дневником

Эти скрипты предоставляют возможности автоматизации работы с электронным дневником через Django ORM.

### Поиск ученика в базе

Находит ученика в базе данных по имени. Если искомый ученик только один, возвращает объект Schoolkid. При отсутствии совпадений останавливает работу и сообщает об ошибке. Если в базе есть несколько совпадений, так же останавливает работу и просит уточнить имя.

### Исправление оценок ученика

Автоматически исправляет все оценики ниже '4' на '5'. Остальные оценки оставляет.

### Удаление замечаний ученика

Удаляет все замечания поставленные указанному учеику.

### Добавление похвалы ученику

Добавляет похвалу от учителя для случайно выбранного урока из вернувшегося списка уроков Музыки в 6А классе.

##### Урок, класс и текст похвалы можно менять в зависимости от нужного предмета.

## Как использовать скрипт

Для начала необходимо убедится что Django окружение настроено правильно.
Что бы запустить Django shell в номом окне коммандной строки вводим ```python manage.py shell```. Надпись ```(InteractiveConsole)``` сообщит нам что Django shell запущен.
Далее импортируем необходимые функции в Django shell. 

Затем найдите ученика по имени коммандой:

```
kid = get_schoolkid(*Имя Ученика*)
``` 

И примените к нему нужные функции:
```
	if kid:
		fix_marks(kid)
		remove_chastisement(kid)
		add_commendation(kid)
```		

#### Используйте с осторожностью - скрипт модифицирует данные в базе.
